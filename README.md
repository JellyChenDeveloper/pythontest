# 安装第三方扩展包

## Scrapy 框架安装
安装该框架需要有`python2.7`,`lxml`,`OpenSSL`

    pip.exe install Scrapy
    
在windows下安装Scrapy时需要有 Microsoft Visual C++ 9.0 环境，在 http://aka.ms/vcpython27 下载。

创建Scrapy项目

    scrapy startproject tutorial
